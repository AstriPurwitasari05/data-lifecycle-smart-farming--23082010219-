# 🌱 Data Lifecycle Smart Farming

## 📋 Deskripsi
Dashboard monitoring kelembaban tanah menggunakan dataset Plant Vase
dari Kaggle: amirmohammdjalili/soil-moisture-dataset

## 📁 Struktur File
- `plant_vase1.CSV` — Raw data sensor vase 1
- `plant_vase1(2).CSV` — Raw data sensor vase 1 (2)
- `plant_vase2.CSV` — Raw data sensor vase 2
- `cleaned_data.csv` — Data setelah proses cleaning
- `streamlit_app.py` — Aplikasi dashboard Streamlit

## 📊 Visualisasi
- Correlation Heatmap per sumber
- Time Series Trend moisture
- Gauge Meter kelembaban
- Alert System threshold

## 🛠️ Teknologi
- Python, Pandas, Streamlit, Plotly, Seaborn
- Google Colab, Kaggle API, ngrok

## 📅 Rentang Data
- Mulai: 06/03/2020
- Akhir: 29/03/2020
- Total: 18,349 baris (setelah cleaning)
